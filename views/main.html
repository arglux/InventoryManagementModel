Getting Started

Before utilising the tool, please format the data into a simple specified format which are either .csv or .xlsx (Excel) files. You may download a sample file here:

Download

Each of the columns represents a quantity that needs to be labeled using a one-letter keyword. Below is an explanation of each column required:

  • Column A (Labeled 'x') - Inventory Level
  • Column B (Labeled 'y') - Units Sold/Demand
  • Column C (Labeled 'Q') - Order Quantity Received
  • Column D (Labeled 'I') - Units in Inventory
  • Column E (Labeled 'B') - Units Backordered
